# 메모장 애플리케이션 Node.js 서버 코드

> 메모장 앱의 백엔드 API를 제공하는 Node.js 코드 레포지토리입니다.
> (인프런 강의에서 제공되는 서버)

## 설치 방법

```
# 패키지 설치
npm install

# 개발 서버 실행
npm run dev
```

(**node v10버전으로 변경후 사용)
